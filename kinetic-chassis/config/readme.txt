Kinetic Chassis JSON File Requirement

0. Each chassis json file must contain at least one chassis "id" field.

1. Each chassis "id" field must be unique.

2. Each chassis must contain at least one "deviceId" field.

3. Each devideId field must contain the required "ips" field with minimum one IP value set in the field.

4. All other fields are optional for the initial phase. 

Please see chassis_template.json for sample contents of the json file.
