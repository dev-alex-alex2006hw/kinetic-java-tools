How to run a kinetic-console demo locally:
==================================
1. mvn clean package

2. cd kinetic-console/bin

3. ./startMultiSimulators.sh

4. ./startKineticConsole.sh

5. open "http://localhost:8080/kinetic/console" in browser


Start rest service steps:
===================================

1. mvn clean package

2. cd kinetic-console/bin

3. ./startRestService.sh
